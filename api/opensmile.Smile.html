SmileÂ¶

class opensmile.Smile(feature_set=FeatureSet.ComParE_2016, feature_level=FeatureLevel.Functionals, *, options=None, loglevel=2, logfile=None, sampling_rate=None, channels=0, mixdown=False, resample=False, segment=None, keep_nat=False, num_workers=1, multiprocessing=False, verbose=False)[source]Â¶

OpenSMILE feature extractor.

  1. You can choose a pre-defined feature set by passing one of opensmile.FeatureSet

  2. You can also provide a custom config file using the following template:

    [componentInstances:cComponentManager]
instance[dataMemory].type=cDataMemory

;;; default source
\{\cm[source{?}:source include config]}

;;; add components reading from reader.dmLevel=wave

;;; combine features
[componentInstances:cComponentManager]
instance[funcconcat].type=cVectorConcat
[funcconcat:cVectorConcat]
reader.dmLevel = <feat-1>;<feat-2>;...
writer.dmLevel = features
includeSingleElementFields = 1

;;; default sink
\{\cm[sink{?}:include external sink]}

Note

The following arguments are not serialized:

  • keep_nat

  • loglevel

  • logfile

  • num_workers

  • multiprocessing

  • segment

  • verbose

For more information see section on hidden arguments.

Parameters

  • feature_set (Union[str, FeatureSet]) â default feature set or path to a custom config file

  • feature_level (Union[str, FeatureLevel]) â default feature level or level name if a custom config file is used

  • options (Optional[dict]) â dictionary with optional script parameters

  • loglevel (int) â log level (0-5), the higher the number the more log messages are given

  • logfile (Optional[str]) â if not None log messages will be stored to this file

  • sampling_rate (Optional[int]) â sampling rate in Hz. If None it will call process_func with the actual sampling rate of the signal.

  • channels (Union[int, Sequence[int]]) â channel selection, see audresample.remix()

  • mixdown (bool) â apply mono mix-down on selection

  • resample (bool) â if True enforces given sampling rate by resampling

  • segment (Optional[Segment]) â when a audinterface.Segment object is provided, it will be used to find a segmentation of the input signal. Afterwards processing is applied to each segment

  • keep_nat (bool) â if the end of segment is set to NaT do not replace with file duration in the result

  • num_workers (Optional[int]) â number of parallel jobs or 1 for sequential processing. If None will be set to the number of processors on the machine multiplied by 5 in case of multithreading and number of processors in case of multiprocessing

  • multiprocessing (bool) â use multiprocessing instead of multithreading

  • verbose (bool) â show debug messages

Examples: >>> sampling_rate = 16000 >>> signal = np.zeros(sampling_rate) >>> smile = Smile( â¦ feature_set=FeatureSet.ComParE_2016, â¦ feature_level=FeatureLevel.Functionals, â¦ ) >>> smile.process_signal(signal, sampling_rate).audspec_lengthL1norm_sma_range start end 0 days 0 days 00:00:01 0.0 Name: audspec_lengthL1norm_sma_range, dtype: float32

Smile.__call__(signal, sampling_rate)[source]Â¶

Apply processing to signal.

This function processes the signal without transforming the output into a pd.DataFrame. Instead it will return the raw processed signal. However, if channel selection, mixdown and/or resampling is enabled, the signal will be first remixed and resampled if the input sampling rate does not fit the expected sampling rate.

  • sampling_rate (int) â sampling rate in Hz

  • RuntimeError â if channel selection is invalid

process_index()Â¶

Smile.process_index(index, *, preserve_index=False, root=None, cache_root=None)Â¶

Extract features from an index conform to audformat.

If cache_root is not None, a hash value is created from the index using audformat.utils.hash() and the result is stored as <cache_root>/<hash>.pkl. When called again with the same index, features will be read from the cached file.

Parameters

  • index (Index) â index with segment information

  • preserve_index (bool) â if True and audinterface.Feature.process.segment is None the returned index will be of same type as the original one, otherwise always a segmented index is returned

  • root (Optional[str]) â root folder to expand relative file paths

  • cache_root (Optional[str]) â cache folder (see description)

Raises

  • RuntimeError â if sampling rates do not match

  • RuntimeError â if channel selection is invalid

  • RuntimeError â if multiple frames are returned, but win_dur is not set

  • ValueError â if index is not conform to audformat

Return type

DataFrame

Smile.to_dict(*, include_version=True, flatten=False, root=None)Â¶

Converts object to a dictionary.

Includes items from audobject.Object.arguments. If an argument has a resolver, its value is encoded. Usually, the object can be re-instantiated using audobject.Object.from_dict(). However, if flatten=True, this is not possible.

Parameters

  • include_version (bool) â add version to class name

  • flatten (bool) â flatten the dictionary

  • root (Optional[str]) â if file is written to disk, set to target directory

Return type

Returns

Examples

>>> import audobject.testing
>>> o = audobject.testing.TestObject('test', point=(1, 1))
>>> o.to_dict(include_version=False)
