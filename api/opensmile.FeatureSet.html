FeatureSet¶

class opensmile.FeatureSet(value)[source]¶

Enumeration of standard feature sets.

The following feature sets are available:

  • ComParE_2016

  • GeMAPS, deprecated alias for GeMAPSv01a

  • GeMAPSv01a

  • GeMAPSv01b

  • eGeMAPS, deprecated alias for eGeMAPSv01a

  • eGeMAPSv01a

  • eGeMAPSv01b

  • eGeMAPSv02

  • emobase

For references, see the papers on:

  • ComParE 2016

  • GeMAPS

  • eGeMAPS

  • emobase
