FeatureLevel¶

class opensmile.FeatureLevel(value)[source]¶

Enumeration of standard feature levels.

  • LowLevelDescriptors - low-level descriptors (LLD) calculated over a sliding window

  • LowLevelDescriptors_Deltas - Delta regression of LLDs

  • Functionals - statistical functionals mapping variable series of LLDs to static values

For more information see https://mediatum.ub.tum.de/doc/1082431/1082431.pdf
